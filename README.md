# Data Science Practices

This is a project to perform data science activities on data using Python and some helper modules of the language like Numpy, Pandas, Matplotlib, etc.

Refer to the [`requirements.txt`](https://github.com/Rakib1508/data-science/requirements.txt) file to get all the necessary modules used for this project. The steps to build the project will be mentioned below as the project develops.

## Project timeline

Here I have documented each steps that I followed to build this project along with date & time when I have completed a step in the project timeline.

#### <time datetime="2021-07-16 00:16:00">July 21, 2021 - 00:16</time>

1. Start the project with a virtual environment. To create the virtual environment, run the given command in any command line terminal.
   `virtualenv ds`
   Activate virtual environment by running
   `ds\scripts\activate` in Windows, or `source bin/activate` in Mac.

2. Install all necessary Python modules given in the `requirements.txt` by running
   `pip install -r requirements.txt`
